Проект разработам с помощью IntelliJ IDEA и фреймворка netty.
Для запуска сервера необходимо запустить файл Server. Сервер слушает по адресу 127.0.0.1 (localhost) порт 8080.
По запросу /hello отдаёт Hello World через 10 секунд,
по запросу /redirect?url=<url> перенаправляет на указанный url,
по запросу /status показывает статистику соединений,
по всем остальным запросам показывает welcome page с предложением перейти на один из трёх вышеуказанных url.
